# Catalyx Property System - UI Mockups

Beautiful, modern UI mockups for the Catalyx Property System based on your reference screenshots.

## 📁 Files

- **login.html** - Login page with dark theme and gradient effects
- **dashboard.html** - Full dashboard with sidebar navigation and stats

## 🚀 How to View

Simply open the HTML files in your browser:

### Option 1: Double-click
- Double-click `login.html` or `dashboard.html` in File Explorer

### Option 2: Browser
- Right-click → Open with → Choose your browser (Chrome, Edge, Firefox)

### Option 3: VS Code Live Server
- Install "Live Server" extension in VS Code
- Right-click HTML file → "Open with Live Server"
